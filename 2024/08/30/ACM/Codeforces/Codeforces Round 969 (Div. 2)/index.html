A.Dora’s Set

注意到,两个偶数肯定不能一起选,一定是2奇+1偶

然后可以暴力打表发现相邻的奇数一定互质,所以答案就是奇数数量/2

B.Index and Maximum Value

C.Dora and C++

发现是个形如exgcd的东西,可以求出最小的变化量为$gcd(a,b)$

将数组的其他数对$gcd$取模,排序,答案最大为$max-min$

再枚举前缀加上gcd

D.Iris and Game on the Tree

注意到,叶子节点的值,只跟根节点和叶子自己有关,具体为

  1. 1(根) 1(叶)=0
  2. 0(根) 0(叶)=0
  3. 0(根) 1(叶)=-1
  4. 1(根) 0(叶)=1

L.502 Bad Gateway

因为每次$roll$的期望是一样的,所以一定是在大于等于某个值的时候$roll$

打表还可以发现,在某个值往后,期望是一样的

设这个值为$x$,有两种做法

1.三分

推导出一个与$x$有关的式子,发现是一个对勾函数,直接三分凹函数最小值就是答案

前$x$的期望
$$\frac{x \ast (1 + x)}{2x}$$

后$T-x$的期望
$$f = 1 + \frac{x}{T}\frac{\ast \left( x \ast (1 + x) \right)}{2x} + \frac{T - x}{T}f$$

移项
$$\begin{aligned}
\frac{x}{T}f &= \frac{ \left( 2T + x + x^{2} \right)}{2T}
\end{aligned}$$

$$\begin{aligned}
f &= \frac{\left( 2T^{2} + xT + Tx^{2} \right)}{2x}
\end{aligned}$$

总期望
$$\frac{\frac{x \ast (1 + x)}{2}\frac{+ \left( 2T^{2} + xT + Tx^{2} \right)}{2x}(T - x)}{T}$$

化简
$$\frac{2T - x + x^{2}}{2x}$$
